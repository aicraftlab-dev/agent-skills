This repository contains a collection of specialized Antigravity Skills designed to guide AI agents through the entire lifecycle of building a product.

- **Product Research & Strategy**: Market analysis and MVP scoping.

- **System Architecture**: Foundations, APIs, and data modeling.
- **Backend Development**: Server-side logic and implementation.
- **Frontend Development**: Visual excellence and UI code.
- **Infrastructure & DevOps**: Deployment and CI/CD.
- **Quality Assurance**: Testing and bug hunting.
- **Product Marketing & Launch**: SEO, copywriting, and GTM.
- **Skill Master**: The management skill for creating and refining these tools.

## Installation & Updates

### 1. Global (Available in every folder)
Use this if you want these skills to be active by default on your machine:
```bash
/Users/alan.reynoso/Projects/skills/bin/ag-skills.sh global
```

### 2. Local Setup (New Project / Repo)
To add these skills to a specific new project directory:
```bash
# Navigate to your new project
cd path/to/your-new-project

# Run the local installer
/Users/alan.reynoso/Projects/skills/bin/ag-skills.sh local
```

### 3. Keep Skills Updated
To pull the latest changes from this repository and update your global/local installs:
```bash
/Users/alan.reynoso/Projects/skills/bin/ag-skills.sh update
```
