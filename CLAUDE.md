# Project Instructions

This project is equipped with the **Product-to-Market Skill Suite**. 

## Agent behavior
- **Skills Usage**: Always prioritize using the specialized skills located in `.agent/skills/`.
- **Phased Approach**: When building from concept to market, follow the sequence defined by the skills:
    1. `product-research-strategy` (Discovery)
    2. `product-design-ux` (Design)
    3. `system-architecture` (Technical Planning)
    4. `backend-development` & `frontend-development` (Implementation)
    5. `quality-assurance-testing` (Verification)
    6. `product-marketing-launch` (GTM)

## Development Workflow
- Keep components modular and reusable.
- Prioritize premium aesthetics and modern design tokens (Glassmorphism, High-quality typography).
- Ensure every feature is backed by a technical plan and a testing strategy.
