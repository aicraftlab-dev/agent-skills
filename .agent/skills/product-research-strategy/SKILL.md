# Product Research & Strategy

This skill guides the agent in transforming a vague concept into a structured product strategy.

## When to use this skill
- When a user provides a new "idea" or "concept".
- When defining the target audience and user personas.
- When performing a competitive analysis.
- When drafting a product roadmap or MVP scope.

## How to use it
1. **Define the Problem Space**: Clearly state the problem the product intended to solve.
2. **User Personas**: Generate at least 2-3 detailed user personas (Name, Role, Pain Points, Goals).
3. **Market Analysis**: Identify potential competitors and unique selling points (USPs).
4. **MVP Scope**: Define the "Must-Haves" for a Minimum Viable Product.
5. **Success Metrics**: Define what success looks like (KPIs).

## Tools & Techniques
- Use `search_web` to find market trends and competitors.
- Use `read_url_content` to analyze competitor websites or documentation.
- Create a `strategy.md` file in the project root to document findings.
